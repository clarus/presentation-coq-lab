\section{More}
\begin{frame}
  \frametitle{More}
  Based on type theory instead of set theory:
  \[
    \{ n \in \mathbb{N}\,|\, \phi(n) \in \mathbb{P}\}
  \]
  \[
    + : \mathrm{nat} \rightarrow \mathrm{nat} \rightarrow \mathrm{nat}
  \]
  \[
    3 + 4 : \mathrm{nat}
  \]

  Reasoning on programming languages made simpler.
\end{frame}
